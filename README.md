## **Build em produção**

Deploy in Heroku [CI/CD] = https://webappnewreact.herokuapp.com/signin

## **Como iniciar ? **

1. Instalar Yarn
`npm install yarn -g`

2. Mapear dependencias
`yarn`

3. Startar Servidor Dev
`yarn start`

4. Rodar Build de Produção
`yarn build`

5. Rodar Suite de Testes
`yarn test`

6. Rodar Linter
`yarn lint`
